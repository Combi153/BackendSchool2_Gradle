# 자바 파일 읽고, SQL 작성하기

## 목표
* 서울시 병의원 위치 정보.csv에서 데이터를 DB에 저장한다.
* 자바에서 쿼리문을 편집한다.
* DB의 table 설정(데이터 타입, 제약 등)에 따른다.

## 소목표
* parser를 인터페이스로 만든다.
* 차후 다른 데이터를 사용할 때도 기존 코드를 재사용할 수 있도록 개발한다.

## 과정
1. 자바에 csv 파일을 불러온다.
2. 행(한 줄)을 읽는다.
3. 파싱한다.
   * district, subdivision 을 편집한다.
4. Hospital 객체에 저장한다.
5. SQL 구문에 맞게 Hospital 객체의 멤버변수를 편집한다.
6. 파일에 SQL 구문을 쓴다.
